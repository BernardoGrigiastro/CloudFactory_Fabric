<h2>Minecraft mod for Fabric</h2>
<p>The mod adds cloud blocks to the game.&nbsp;Cloud blocks you can place anywhere in air.<br /><br /><img src="https://github.com/dmkenza/CloudFactory_Fabric/blob/1.19.2/photos/4.png?raw=true" alt="" width="512" height="288" /><br /><br /><span style="font-size: 18px;"><strong>1) The Cloud Generator.<br /></strong></span><br />You can create clouds by using a new block generator. <br />As fuel for clouding you have to use new item - Alumentum.<br /><img src="https://github.com/dmkenza/CloudFactory_Fabric/blob/1.19.2/photos/8.png?raw=true" alt="Clouding" width="513" height="223" /><br /><br />There are 16 kinds of cloud for each color.<br /><br /><span style="font-size: 18px;"><strong>2)&nbsp;Jewelry.<br /></strong></span><br />Clouds are transparent and you will slow&nbsp;fall down if you dont wear special&nbsp;<span class="Y2IQFc" lang="en">bijouteries from the mod.<br />You can place this items in slots from&nbsp;</span><a href="https://www.curseforge.com/minecraft/mc-mods/trinkets">trinkets </a>mod.</p>
